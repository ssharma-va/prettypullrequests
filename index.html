    A chrome extension to make reviewing pull requests in Github a little easier.

    Project maintained by ssharma-va Hosted on GitHub Pages — Theme by mattgraham

    Pretty Pull Requests

    This extension will add various features to the diff view on Github's pull request pages.

    Supports both files changed and commit diffs.

    Current Features:

    • Click on the file name to collapse a file
    • Collapse/Expand all
    • Collapse/Expand all that match a criteria
    • Collapse/Expand added/removed lines
    • Move the close pull request button slightly away from the comment button

    Collapse Diffs: Simply click anywhere on the path (cursor will turn to the hand).

    Collapse/Expand all, or any path containing a word: Collapse/Expand all with an empty path, or type a word to match any part of the path and collapse/expand any diffs that match. Note: it IS case sensitive.

    Collapse/Expand Additions/Deletions: Note the checkboxes on the top right of the diff that are added to control this feature.

    Moved close pull request button away and gave it a better name: This helps prevent closing the pull request due to annoying misclicks.
